# [Das-Kapital](https://knowpedia.github.io/Das-Kapital)
资本论，包括资本的生产过程、资本的流通过程、资本主义生产的总过程等知识。

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/knowpedia/Das-Kapital/issues)，欢迎参与维护！

## How to use?

开发的时候运行下列命令：

```
npm run dev
```

开发完毕需要发布的时候，请运行下列命令：

```
npm run build
```
